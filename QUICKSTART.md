# Kjs-Nuvio Plugin - Quick Start Guide

## Installation

1. Open **Nuvio** app
2. Go to **Settings → Local Scrapers**
3. Click **Add Repository**
4. Enter this URL:
   ```
   https://raw.githubusercontent.com/KotlinJS5/Kjs/main/
   ```
5. Click **Add**
6. Enable **4KHDHub** provider
7. Close and reopen Nuvio

## Usage

1. Search for any movie or TV show
2. 4KHDHub will appear in the provider list
3. Select streams and play

## Supported Content

- ✅ Movies (all years)
- ✅ TV Shows (with season/episode support)
- ✅ Multiple quality options (HD, 1080p, 4K)

## Troubleshooting

### Plugin doesn't appear
- Clear Nuvio cache (Settings → Storage → Clear Cache)
- Restart Nuvio
- Re-add the repository

### No streams found
- Try searching for popular movies: "The Matrix", "Inception", "Avatar"
- Check your internet connection
- Wait 5-15 seconds for search to complete

### Streams won't play
- Try opening in external player
- Check if URL is accessible from your region
